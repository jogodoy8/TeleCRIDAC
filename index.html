Integração de Equipe Slide 2

Propósito e Impacto

Para o Cidadão

Eliminar barreiras geográficas em Mato Grosso. O TeleCRIDAC leva o especialista até o paciente, garantindo dignidade e acesso especializado via telessaúde humanizada.

Para a Gestão SES-MT

Interoperabilidade e dados em tempo real. Transformamos cada consulta em logs estruturados para faturamento SUS e métricas de eficiência clínica.

Valor do Projeto Slide 3

A Experiência do Profissional

O foco do desenvolvedor é simplificar a vida do fonoaudiólogo, fisioterapeuta e médico. Nossa interface deve permitir:

  • Registro ágil do IFBrM (Avaliação de Deficiência).
  • Sincronização automática com o banco de dados.
  • Visualização do histórico do paciente em um clique.

Foco no Usuário Slide 4

Nossa Tech Stack

GitHub

Versionamento, automação com Actions e segurança via Secrets para proteção de chaves de API.

Supabase

Nosso "Backend as a Service". PostgreSQL em tempo real, Autenticação e Armazenamento (Storage) resiliente.

Python / Web

Linguagem base para processamento de dados e integração de scripts de automação de faturamento e rotinas internas.

Municípios Potenciais

Escalabilidade Total

O TeleCRIDAC foi desenhado para não se limitar à Cuiabá. Ele atua como um hub central, irradiando conhecimento clínico especializado e descentralizando o cuidado para todo o estado de Mato Grosso.

Alcance Estadual Slide 9

Segurança e Governança

  • Conformidade LGPD: Dados sensíveis e prontuários estruturados de forma segura, com controle rígido de acesso.
  • Gestão de Credenciais: Chaves de API e senhas de produção totalmente protegidas nas variáveis do ambiente (GitHub Secrets).
  • Infraestrutura Nacional: Hospedagem configurada na região da América do Sul (São Paulo) para garantir menor latência operacional.
